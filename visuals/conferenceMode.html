Interaction Design for conference mode scenario for students
The purpose of this design was to create the conference mode module in LearniCo Student Mobile App. The student can listen to the teacher and see thier audio/video.
If student likes a specific topic/ presentation slide, student can like it, If student has question, student can ask doubt in the online class to the teacher.
There is provision for class chat where in students can classmates can interact with the teacher and amongst themselves.
I created 3 variations of the design, the 2nd design was implemented as it provided a clean interface and no distractions for the student.
Design 1
The goal of this design was to show student about ongoing chat in the classroom.
Design 2
Here the student can click on like, question and chat. On clicking chat icon, student is redirected to new page where they can see the class chat.
Design 3
This approach was inspired by youtube. But it did not follow hicks law, it was distracting. The main focus was going on the chat which was not required.
