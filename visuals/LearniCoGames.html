Designing Learning Games for LearniCo
I created 4 learning games for LearniCo application that would encourage student participation, learning and performance in the classroom.
I had learned about YUKAI's Principles of Gamification. These principles helped me to apply the analogies to create learning games.
Game 1
I got the idea from the game of Musical Chairs. In this game teacher will have to select the number of questions and time limit for the game.

The students would be randomly selected to solve a question. The question appears on their smartphone. If the student answers the question correctly, the student stays in the game, if not, the student is out!
Game 2 - Jeopardy
This is inspired by the very popular game - Jeopardy. Students can pick up the question hidden behind the points. More points means more difficulty of the question.
Game 3 - Open Ended questions
Teacher can ask any open ended question to students with this game.
Game 4 - Best worst idea
This game was inspired by an ideation technique that I learned in the IDF UX Courses.
Game 5 - Spin the Wheel
Spin the wheel, gives opportunity to solve a question in group. If your is selected, some one from your group needs to ask a quesiton to the entire class!
